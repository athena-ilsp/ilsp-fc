# ILSP Focused Crawler (ILSP-FC)

ILSP Focused Crawler (ILSP-FC) is a research prototype for acquiring domain-specific monolingual and bilingual corpora. 

## Reference

Papavassiliou, V., Prokopidis, P. & G. Thurmair. (2013). [A modular open-source focused crawler for mining monolingual and bilingual corpora from the web.](https://aclweb.org/anthology/W/W13/W13-2506.pdf) In Proceedings of the Sixth Workshop on Building and Using Comparable Corpora, pages 43-51. Sofia, Bulgaria : Association for Computational Linguistics ([BibTeX](https://aclweb.org/anthology/W/W13/W13-2506.bib))
